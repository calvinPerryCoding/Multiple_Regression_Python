Prerequisites
basic understanding of python and pip
pandas, numpy, and sklearn must be installed using pip

About 
This program uses Multiple Regression to predict whether or not a customer will buy something
This program is curently set up for how likely is a customer going to buy Halloween Candy
This program factors in age, gender, time of year, whether or not the customer is a parent, education, salary

Note: The data conatins only 19 entries and is only an example, 
in the real world you would have hundreds or even thousands of entries.
